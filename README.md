# ⏰ SillyTavern 时间感知插件（Time Awareness）

用于 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 让 LLM 获得“当前时间 + 日期氛围 + 天气信息”感知，并在纪念日/久未发言时主动消息。

## 功能亮点
- **时间感知**：自动向 LLM 注入当前时间、时间段（凌晨/早晨/上午/中午/下午/傍晚/夜晚/深夜）、工作日/周末/节假日、距离上次用户发言。
- **节假日识别**：内置接入 `chinese-days`（法定节假日+调休、农历民俗节日），网络不通则自动降级为“工作日/周末”判断。
- **纪念日提示**：自定义纪念日（生日、周年等），当天自动告诉 LLM，并可计算“第 N 年”。
- **自动消息**：
  - 纪念日当天角色主动发消息。
  - 用户长时间未发言时，按概率触发“角色主动问候/闲聊”。
  - 可限定生效的角色名单。
- **天气与空气质量（Open‑Meteo）**：
  - 用户输入城市/区县，自动定位经纬度。
  - 注入当前天气、空气质量、未来预报。
  - 为隐私考虑，注入时不包含地名。
  - 带缓存与定时更新，避免频繁请求。

## 安装方式

### 方式一：SillyTavern 内置安装
- 打开 SillyTavern → Extensions
- 点击 “Install extension”
- 粘贴仓库的克隆 URL
- 点击 “Install”
- 刷新 SillyTavern 页面

### 方式二：手动安装
- 下载仓库压缩包并解压
- 放入以下路径：SillyTavern/public/scripts/extensions/third-party/
- 刷新 SillyTavern 页面

## 基础使用
- **启用插件**：勾选“启用插件”。
- **预览注入内容**：点击“预览当前注入内容”，可看到本轮请求会给 LLM 注入哪些时间信息。
- **纪念日**：添加“名称 + 日期(必填，MM-DD) + 年份(可选，算第几周年)”，当天会提醒 LLM。
- **节假日/CDN 状态**：面板右上角会显示“节假日库 ✓/✗”，若为 ✗ 表示网络取不到节假日数据（仍可正常工作日/周末判断）。

## 天气与空气质量（Open‑Meteo）
- 在“天气与空气质量”区域输入城市/区县并搜索。
- 选中正确地点后，会保存经纬度。
- 勾选“启用天气注入”，即可在提示词中加入天气信息。
- 可自由开关：当前天气 / 空气质量 / 未来预报。
- 可设定预报天数与更新间隔（分钟）。

## 自动消息（可选）
> 注意：前端插件需保持浏览器页面 **打开** 才能定时检查与触发。

- **纪念日主动消息**：启用后，纪念日当天角色会自动生成一条消息插入聊天，看起来像角色主动发的。
- **久未发言主动问候**：
  - “闲置触发阈值（小时）”达到后才会开始滚概率；
  - “到达阈值后检查间隔（分钟）”控制多长时间试一次；
  - “每次检查触发概率（%）”决定是否真触发；
  - 可在“自动消息适用角色”中勾选/留空（留空=对所有角色生效）。
- **手动测试**：点击“手动测试闲置消息”即可立即触发一次生成，验证流程是否正常。

## 推荐默认值
- 闲置触发阈值：4 小时
- 检查间隔：30 分钟
- 触发概率：10%~20%
- 天气更新间隔：30 分钟
- 未来预报天数：3 天

## 常见问题
- **为什么必须开着页面？**
  插件逻辑跑在浏览器前端，关掉页面 JS 就停了，无法定时触发。
- **节假日显示异常/无节日信息？**
  检查 CDN 是否可访问；不可访问时会降级为工作日/周末判断。
- **角色列表为空？**
  请先导入或加载角色卡，刷新插件面板的“🔄”按钮。
- **Open‑Meteo 能否做到秒级实时？**
  Open‑Meteo 提供的是模型“近实时”数据（通常 15 分钟或小时级更新），不是气象站秒级实况。
